Headings Tag
There are six headings available in HTML.
Tag Name Description
<h1> This is Heading 1 The <h1> to <h6> elements represent six levels of headings in a document, <h1> is the most important and <h6> is the least important. A heading element briefly describes the topic of the section it introduces.
Heading information can be used by user agents to construct a table of contents for a document automatically.
<h2> This is Heading 2
<h3> This is Heading 3
<h4> This is Heading 4
<h5> This is Heading 5

Container Tag
There are several container tags in HTML.
Tag Name Description
<div> This is div/division block div tag is used to create blocks or sections in the document.
<span> This is span block span is a container for inline content.
<p> This is a paragraph This is for the paragraph tag.
